Daily Template

~6 Hours/Day · 6 Days/Week

Same shape every day. Stop fighting motivation — let structure do the work.

⏰ 90 MIN · MORNING

Learn the next concept

One topic, one source. Read docs, not courses. No videos >30 min.

⏰ 30 MIN · MID-MORNING

DSA — 1 problem

NeetCode pattern of the day. Time-box 30 min. Read solution if stuck.

⏰ 3 HRS · AFTERNOON

Ship code

Project work only. Push to GitHub every day. No "setup" days.

⏰ 60 MIN · EVENING

From W4: Apply + Outreach

5 applications/day. 3 cold DMs. 1 LinkedIn/X post weekly.

Week-by-Week

The 6 Weeks

Tick each "Ship" item — that's what drives your sprint progress. Skip the "learn" if you already know it; you don't get bonus points for re-reading.

WEEK 1 · DAYS 1–6

Backend Crash: Node + Postgres + Auth

~36 hrs · Lean on your JS/TS
Learn (compressed)
  • Node event loop, async, streams (1 day)
  • Hono or Express + Zod validation
  • Postgres + Prisma (schema → migrate → query)
  • JWT + refresh tokens · Clerk for speed
  • Deploy: Fly.io or Railway
✓ Ship: Auth + CRUD API deployed publicly. Postman/Bruno collection in repo.
WEEK 2 · DAYS 7–12

Next.js Full-Stack → Ship Project 1

~36 hrs · Your React experience compounds here
Learn (compressed)
  • Next 15 App Router: RSC, Server Actions, route handlers
  • Caching layers + streaming + Suspense
  • tRPC for type-safe API (skip GraphQL)
  • Tailwind + shadcn/ui (you already know CSS)
  • Sentry + posthog wired in 30 min
✓ Ship: Project 1 — Pulse live with auth, dashboard, deployed. README + 60s Loom.
WEEK 3 · DAYS 13–18

LLM Engineering + RAG

~36 hrs · The AI core
Learn (compressed)
  • Anthropic SDK + Vercel AI SDK (streaming, tool calling)
  • Prompt patterns: structured output, JSON schemas, CoT
  • Embeddings + pgvector (one DB, no Pinecone yet)
  • RAG pipeline: chunk → embed → retrieve → re-rank → cite
  • Inngest for async ingestion jobs
✓ Ship: RAG prototype inside Project 2 scaffold — answers with citations. Deploy preview live.
WEEK 4 · DAYS 19–24

Agents + MCP + Evals · 🚀 START APPLYING

~36 hrs · Differentiator week
Learn (compressed)
  • Tool calling: function schemas, parallel calls, streaming
  • Agent loops: ReAct + Plan-Execute (use Vercel AI SDK or Mastra)
  • Build 1 MCP server (stdio) exposing real tools
  • Langfuse or Braintrust for traces + evals
  • Prompt injection basics + guardrails
✓ Ship: Agent feature inside Project 2 (tool calling + traces). Send first 25 applications.
WEEK 5 · DAYS 25–30

Finish Project 2 + Interview Loop

~36 hrs · Apply + interview in parallel
Build & convert
  • Polish Project 2: streaming UI, eval suite, cost dashboard
  • Architecture diagrams (Excalidraw) for both projects
  • STAR stories: 3 per project, written down
  • 1 system design dry run (AI chat product)
  • Daily: 5 apps + 3 cold DMs + take-home if received
✓ Ship: Project 2 — Lexicon live + portfolio site + 2 blog posts published.
WEEK 6 · DAYS 31–36

Polish, Convert, Negotiate

~36 hrs · Close offers
Convert mode
  • Crush onsites — 2 mock interviews this week (Pramp/Exponent)
  • Fix any rough edge in projects flagged by interviewers
  • 1 OSS PR to an AI tool you use (Vercel AI SDK, Mastra, etc.)
  • Negotiation prep: ranges by stack, market data
  • If no offer yet → continue into weeks 7–8 buffer
✓ Ship: Offer received OR clear pipeline of 3+ final rounds for weeks 7–8.
Weeks 7–8 (buffer, optional): Only if Week 6 doesn't close. Don't pre-plan it — earning the buffer is fine, planning to use it leaks urgency.

Hard Skips

SKIP: Kubernetes, Terraform, Docker Swarm. Vercel + Fly is enough.
SKIP: Microservices, Kafka, gRPC, event sourcing.

SKIP: Heavy DSA (300+ leetcode). 25 problems is your budget.
SKIP: Third flagship project. Polish two instead.
SKIP: Custom design systems. Tailwind + shadcn — done.

SKIP: Watching YouTube full-stack courses. You're past that.
